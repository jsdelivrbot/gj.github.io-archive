Remember... 
play_pattern_timed [F-pattern], [0.5]
play_pattern_timed [E-pattern], [0.5]
...is the same as... 
play_pattern_timed [F-pattern, E-pattern], [0.5]
...which is the same as... 
play_pattern_timed [F-pattern,
                    E-pattern], [0.5]

From 2,272 to 74 to 33 to 22 lines of code already. Not bad! Now try substituting the corresponding play_pattern_timed methods in for the pseudocode, and then hit Run. How does it sound? If something feels off, check your code against mine, hidden below the image of Jake cooking.
